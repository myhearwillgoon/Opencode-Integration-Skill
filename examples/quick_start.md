# opencode-integration - 快速开始

## 基本用法

这个技能从 /home/guanhb/.openclaw/workspace/skills/opencode-integration 迁移而来。

## 安装

```bash
# 确保在技能目录中
cd /path/to/opencode-integration

# 运行安装脚本
./install.sh
```

## 配置

1. 查看 `config_templates/` 目录中的配置示例
2. 根据需要调整配置
3. 运行测试验证安装

```bash
./test_installation.sh
```
